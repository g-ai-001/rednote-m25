# 项目规划

## 项目概述
- **项目名称**: rednote-m25 (仿小红书)
- **项目类型**: Android本地单机应用
- **版本**: 0.1.0

## 功能规划

### 短期规划 (0.1.x)
#### 0.1.0 - 基础框架搭建
- [x] 搭建项目基础框架
- [ ] 首页瀑布流笔记展示
- [ ] 笔记详情页
- [ ] 图片浏览功能
- [ ] 本地数据存储
- [ ] 日志系统

### 中期规划 (0.2.x - 0.3.x)
- 笔记发布功能
- 收藏功能
- 分类浏览
- 搜索功能
- 用户个人主页

### 长期规划 (0.4.x+)
- 笔记编辑功能
- 话题功能
- 笔记点赞功能
- 笔记评论功能
- 数据导出/导入

## 技术架构
- **架构模式**: MVVM + Clean Architecture
- **UI框架**: Jetpack Compose + Material 3
- **依赖注入**: Hilt
- **本地存储**: Room + DataStore
- **图片加载**: Coil
- **异步处理**: Kotlin Coroutines + Flow
