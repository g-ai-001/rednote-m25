# rednote-m25

仿小红书安卓应用，本地单机使用。

## 功能特性

- **首页瀑布流**：展示笔记列表
- **笔记详情**：查看笔记内容、图片、作者信息
- **笔记发布**：支持发布新笔记
- **笔记编辑**：支持编辑已有笔记
- **笔记删除**：支持删除我的笔记
- **笔记评论**：支持查看和添加评论
- **分类浏览**：按标签分类浏览笔记
- **话题探索**：发现页展示热门话题
- **个人主页**：查看我的笔记和收藏
- **用户头像**：支持设置用户头像
- **数据导出**：支持导出笔记和评论为JSON文件
- **数据导入**：支持从JSON文件导入数据
- **点赞收藏**：支持点赞和收藏笔记
- **搜索功能**：支持标题和内容关键词搜索
- **我的收藏**：查看已收藏的笔记列表
- **本地存储**：使用Room数据库存储数据
- **精美主题**：Material 3设计语言，红色主题

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- Coil 图片加载
- Navigation Compose
