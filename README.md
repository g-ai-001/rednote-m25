# rednote-m25

仿小红书安卓应用，本地单机使用。

## 功能特性

- **首页瀑布流**：展示笔记列表，支持下拉刷新
- **笔记详情**：查看笔记内容、图片、作者信息
- **点赞收藏**：支持点赞和收藏笔记
- **本地存储**：使用Room数据库存储数据
- **精美主题**：Material 3设计语言，红色主题

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- Coil 图片加载
- Navigation Compose
